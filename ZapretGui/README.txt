# Zapret Matrix GUI - Portable версия

## Быстрый старт

1. Запустите **ZapretMatrixGUI.exe**
2. Выберите стратегию (например, general.bat)
3. Нажмите "■ АКТИВАЦИЯ ■" для запуска
4. Нажмите "■ ДЕАКТИВАЦИЯ ■" для остановки
5. Готово ✨

## Управление

### Кнопки
- **■ АКТИВАЦИЯ ■** (зелёная) - запускает выбранную стратегию
- **■ ДЕАКТИВАЦИЯ ■** (красная) - останавливает сервис и процессы
- **Обновить статус** - проверяет текущее состояние

### Что делает кнопка деактивации?
1. Останавливает сервис через `service.bat uninstall`
2. Принудительно завершает процесс `winws.exe`
3. Проверяет что всё остановлено
4. Выводит результат в консоль

## Требования

- Windows 10/11
- Права администратора (для запуска сервиса)
- **Python НЕ нужен**

## Содержимое

- `ZapretMatrixGUI.exe` - основная программа
- `service.bat` - сервис запрета
- `general*.bat` - стратегии обхода
- `bin/` - системные файлы WinDivert
- `lists/` - списки доменов и IP

## Документация

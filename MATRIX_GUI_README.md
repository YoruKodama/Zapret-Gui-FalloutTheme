# 🎮 ZAPRET PIP-BOY GUI 🎮

## Визуальный интерфейс в стиле Fallout Pip-Boy 3000

Профессиональное десктопное приложение для управления Zapret с ретро-футуристическим интерфейсом в стиле терминалов Fallout.

---

## 🚀 БЫСТРЫЙ СТАРТ

```bash

```

### 2. Запуск
```bash

```

Или через главное меню:
```bash

```

## ⚙️  ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Основано на:
- **CustomTkinter** — современный UI фреймворк
- **Python 3.8+** — основной язык
- **psutil** — мониторинг процессов
- **threading** — асинхронные операции

### Цветовая схема Matrix:
```
MATRIX_GREEN = #00FF41  (основной зелёный)
DARK_GREEN   = #003B00  (тёмно-зелёный)
BLACK        = #0D0208  (чёрный фон)
RED          = #FF0000  (ошибки/остановка)
YELLOW       = #FFD700  (предупреждения)
```

### Размер окна:
- 1000x700 пикселей
- Фиксированный (не изменяется)
- Центрируется на экране

## 🎮 ИСПОЛЬЗОВАНИЕ

### Базовый сценарий:
1. Запустить `run_matrix_gui.bat`
2. Выбрать стратегию из списка
3. Нажать **▶ ЗАПУСТИТЬ**
4. Смотреть статус → `● АКТИВЕН` (зелёный)
5. Проверить работу интернета

### Смена стратегии:
1. Нажать **⏹ ОСТАНОВИТЬ**
2. Выбрать другую стратегию
3. Нажать **▶ ЗАПУСТИТЬ**

### Диагностика:
1. Нажать **🧪 Диагностика**
2. Откроется отдельное окно
3. Следовать инструкциям

### Тестирование:
1. Нажать **✓ Тесты стратегий**
2. Откроется окно тестов
3. Выбрать типы тестов

## ❓ FAQ

**Q: Приложение не запускается**
```
A: Установите зависимости:
   install_dependencies.bat
```

**Q: "Python не найден"**
```
A: Установите Python 3.8+
   https://www.python.org/downloads/
   ✓ Отметьте "Add Python to PATH"
```

**Q: Статус "ОСТАНОВЛЕНО" но интернет работает**
```
A: Это нормально, winws.exe может быть запущен вручную
```

**Q: После запуска интернет не работает**
```
A: 1. Пробуйте другие стратегии (ALT версии)
   2. Используйте Диагностику
   3. Перезапустите браузер
```

**Q: Как включить автозапуск?**
```
A: 1. Отметьте "Автозапуск при старте"
   2. Установите службу через service.bat
```

### Ошибка импорта CustomTkinter
```bash
pip install --upgrade customtkinter
```

### Ошибка импорта psutil
```bash
pip install psutil
```

### Ошибка "winws.exe не найден"
```
Убедитесь что:
✓ Находитесь в папке zapret
✓ Папка bin/ содержит winws.exe
✓ Файл не заблокирован антивирусом
```

```
1. Проверьте антивирус (добавьте исключение)
2. Запустите от имени администратора
```
